¿Qué es una red?

Una red es un grupo de dispositivos, como computadoras y teléfonos, que están conectados entre sí para poder comunicarse y compartir información. Imagina una red como un sistema de caminos donde los datos pueden viajar de un lugar a otro. Por ejemplo, cuando envías un mensaje a un amigo, este viaje ocurre a través de una red. Las redes pueden ser pequeñas, como las que se usan en casa, o grandes, como Internet.

Clases de redes

  • Red de Área Local (LAN): Conecta dispositivos en un área pequeña, como una casa o una oficina. Es rápida y permite compartir recursos, como impresoras.
  • Red de Área Amplia (WAN): Cubre un área más grande, como ciudades o países. Conecta múltiples LANs. Internet es el ejemplo más grande de una WAN.
  • Red de Área Metropolitana (MAN): Une varias LANs en una ciudad o una región. Es más grande que una LAN pero más pequeña que una WAN.
  • Red Personal (PAN): Conecta dispositivos personales cercanos, como teléfonos, tabletas y computadoras, normalmente en un rango de pocos metros.
  • Red Inalámbrica (WLAN): Es una LAN que no usa cables, permitiendo que los dispositivos se conecten a través de señales de radio, como Wi-Fi.
  • Red Virtual Privada (VPN): Crea una conexión segura a través de Internet, permitiendo que los usuarios accedan a una red privada desde cualquier lugar.

Estas son las principales clases de redes, cada una diseñada para cumplir diferentes necesidades de comunicación y conexión.

¿Qué es un switch?

Un switch es un dispositivo de red que conecta varios dispositivos dentro de una misma red local (LAN). Su función principal es recibir datos de un dispositivo y enviarlos al dispositivo de destino correcto dentro de la red, utilizando direcciones MAC para identificar los dispositivos conectados.

Características principales

  • Conmutación de paquetes:Los switches operan en la capa 2 (enlace de datos) del modelo OSI, lo que significa que manejan las direcciones MAC para conmutar tramas de datos entre dispositivos en la red.
  • Aprendizaje de direcciones: Un switch mantiene una tabla de direcciones MAC, donde almacena la dirección MAC de cada dispositivo conectado y el puerto al que está conectado. Esto permite al switch aprender dinámicamente dónde se encuentran los dispositivos en la red.
  • Colisión de datos: A diferencia de un hub, que envía datos a todos los puertos, un switch envía datos solo al puerto específico al que está conectado el dispositivo de destino, lo que minimiza las colisiones y mejora la eficiencia de la red.
  • Velocidad: Los switches suelen ser más rápidos que los hubs porque procesan datos en hardware y pueden realizar múltiples transmisiones simultáneamente.
  • Configurabilidad: Algunos switches son "inteligentes" o "gestionados", lo que significa que permiten configuraciones avanzadas, como la creación de VLANs (redes locales virtuales), gestión de tráfico y control de acceso.

Usos comunes

  • Conectar computadoras, impresoras y otros dispositivos en una oficina o en el hogar.
  • Expansión de redes existentes.
  • Segmentación de redes para mejorar el rendimiento y la seguridad.

Modelo OSI

El modelo OSI (Interconexión de Sistemas Abiertos) es un marco teórico que describe cómo se comunican los sistemas de red. Está dividido en siete capas, cada una de las cuales tiene una función específica en el proceso de comunicación. Las capas, de arriba hacia abajo, son:

  1. Capa de Aplicación: Interactúa con el usuario y las aplicaciones.
  2. Capa de Presentación: Formatea y traduce datos para la aplicación.
  3. Capa de Sesión: Establece, mantiene y finaliza conexiones entre aplicaciones.
  4. Capa de Transporte: Asegura la entrega de datos y controla el flujo.
  5. Capa de Red: Determina la ruta que seguirán los datos a través de la red.
  6. Capa de Enlace de Datos: Proporciona transferencia de datos entre dispositivos en la misma red.
  7. Capa Física: Transmite los datos en forma de señales eléctricas o luz.

Cableado estructurado

El cableado estructurado es un sistema estandarizado para instalar y organizar cables en un edificio o instalación. Está diseñado para soportar diferentes tipos de redes (como datos, voz y video) de manera eficiente, ordenada y flexible. Incluye cables, conectores, paneles de parcheo y otros componentes que permiten la interconexión de dispositivos de red y asegura que el sistema pueda ser fácilmente gestionado y actualizado.

Ejemplo de Cableado Estructurado

Imagina una oficina de varios pisos que necesita conectar computadoras, teléfonos, cámaras de seguridad y servidores a una red central.

Sala de telecomunicaciones: En cada piso hay un cuarto específico donde se concentran los cables que conectan todos los dispositivos de ese piso. Estos cuartos contienen paneles de parcheo (patch panels) y switches de red.

Backbone: Desde cada sala de telecomunicaciones, un conjunto de cables de alta capacidad (como fibra óptica o cables de par trenzado) conecta los cuartos de cada piso a una sala de servidores central en el sótano, formando la columna vertebral de la red.

Cables horizontales: Desde la sala de telecomunicaciones de cada piso, cables de par trenzado (como Cat 6 o Cat 6a) se extienden a cada estación de trabajo, conectando computadoras, teléfonos IP y otros dispositivos.

Paneles de parcheo y racks: Los cables de cada dispositivo terminan en un panel de parcheo en la sala de telecomunicaciones, lo que permite conectar y organizar fácilmente las conexiones entre dispositivos y la red.

Tomas de red: En cada estación de trabajo hay tomas de red (puertos RJ45) que permiten conectar dispositivos a la red mediante cables Ethernet.
